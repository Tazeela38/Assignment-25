# Internship Application Portal

## Setup Instructions:
1. Import `sql/internship_portal.sql` to your MySQL.
2. Update DB credentials in `includes/db.php`.
3. Place project folder in `htdocs` (XAMPP) or your web server root.
4. Visit `index.php` to test user form.
5. Go to `admin/login.php` to test admin.

## Default Admin:
Username: admin
Password: [hash this using PHP's password_hash()]

## Features:
- File validation and upload
- Session-secured admin
- Status update functionality
